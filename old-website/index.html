Hi there, I'm an aspiring software developer. I'm currently an undergraduate at Amherst College. My programming experience ranges from basic Java applets to iOS game development to full stack web programming. There are still a lot of programming and computer science domains I want to explore: systems programming, desktop game development, machine learning, etc. to name a few. I'm also constantly trying to be aware of good design practices. If there is such a thing as a renaissance programmer, I want to become one.

Besides programming, I spend most of my free time reading. I read mostly science fiction and contemporary novel books. My favorite author is Haruki Murakami. Please add me on Goodreads. I'm always looking for a good book to read. My other interests include: manga, anime, drawing, movies and running.
